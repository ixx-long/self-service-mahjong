# TabBar 图标说明

需准备以下 6 张 40×40px 的 PNG 图标放入此目录：

| 文件 | 用途 |
|------|------|
| `tab-home.png` | 首页（未选中） |
| `tab-home-active.png` | 首页（选中·金色） |
| `tab-order.png` | 订单（未选中） |
| `tab-order-active.png` | 订单（选中·金色） |
| `tab-mine.png` | 我的（未选中） |
| `tab-mine-active.png` | 我的（选中·金色） |

> 可使用 iconfont 或 Figma 导出。选中态图标建议使用主题色 `#D4AF37`。
>
> 开发阶段可使用微信开发者工具自带图标临时替代（修改 `app.json` 中的 `iconPath` 为 `""` 即可使用默认图标）。
